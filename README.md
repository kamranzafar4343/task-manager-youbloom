# Laravel Task Manager

A simple Task Manager application built with **Laravel 12**, **Bootstrap 4**, and **MySQL**.

## Features

* Create, View, Edit, and Delete tasks
* Task status (Pending / Completed)
* Bootstrap 4 responsive UI
* Form validation
* Delete confirmation modal

## Installation

```bash
composer install
cp .env.example .env
php artisan key:generate
```

Configure your database in `.env`, then run:

```bash
php artisan migrate
php artisan serve
```

Visit: `http://127.0.0.1:8000`

## Tech Stack

* Laravel 12
* PHP 8.2
* Bootstrap 4
* MySQL
